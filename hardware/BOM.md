# Bill of Materials (Low-cost prototype)
Recommended low-cost components for pilot deployment (price estimates in INR):

- Raspberry Pi 4 (2GB) - ₹4,500
- ESP32 dev board - ₹400
- Cheap USB webcam or IP camera - ₹1,500
- HC-SR04 Ultrasonic sensor - ₹120
- IR break-beam sensors - ₹80
- LoRa SX1278 module (pair) - ₹600
- GPS NEO-6M module - ₹700
- LED matrix / signage (small) - ₹800
- 12V power supply + battery backup - ₹1,000

**Estimated pilot cost (per intersection): ~ ₹9,000 - ₹12,000**

Notes:
- Use existing CCTV cameras where available to reduce cost.
- ESP32 + LoRa are used for low-power remote sensors and traffic light controllers.
